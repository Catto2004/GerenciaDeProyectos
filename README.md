# Proyecto de Gestión de Talento Humano

Prototipo mínimo en Python + SQLite (consola). Credenciales por defecto:

Estructura creada:

- `screens/` - menús y flujos (Login, Dashboard, Candidatos, Contratos, Reportes).

Ejecutar:

