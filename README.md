# Proyecto de Gestión de Talento Humano

Aplicación de consola en Python + SQLite para administrar candidatos, contratos y reportes.

## Acceso

- Usuario: `admin`
- Password: `1234`

## Menú principal

1. Gestionar candidatos
2. Generar contrato
3. Reportes
4. Desarrollador
5. Manual de usuario
6. Salir

## Desarrollador

- Limpiar BD: elimina candidatos y contratos, dejando el sistema en cero.
- Cargar ejemplo: crea 10 candidatos y 4 contratos de ejemplo.

## Ejecutar

```powershell
python App.py
```
